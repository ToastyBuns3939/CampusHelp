During the event period, this is an event-only production that can only be attempted by eligible idols.

You can produce in event mode by turning on event mode from production.

In event mode, regardless of PLv, skill cards and P drinks unlocked by PLv will appear during production, and memories can also be rented regardless of PLv.

※Skill cards unlocked by PLv acquired during event mode will be included in the memory lottery.

In event mode, you can start producing with event P items in your possession, and P idol-specific skill cards and event-specific P drinks will appear during production.

The evaluation method for producing is different from normal producing, and "Starlight" will affect the production evaluation.

※The P idol-specific skill cards that appear will differ depending on the plan.
※Production evaluation in event mode is not included in the production ranking.
