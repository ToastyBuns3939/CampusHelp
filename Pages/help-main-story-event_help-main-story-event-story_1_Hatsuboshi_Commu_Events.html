The Hatsuboshi Commu Event is an event where you can watch Hatsuboshi Commu stories.

By completing the daily "Hatsuboshi Commu Event Limited Missions," you earn mission points. Collecting mission points unlocks Hatsuboshi Commu stories.

Also, by producing idols or completing event missions, you can earn event points and event exchange items. Collecting event points grants various rewards. Event exchange items can be exchanged for various items at the "Exchange Shop."
