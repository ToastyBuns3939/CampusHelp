If the idol who acquired the Memory has achieved a True End, by using the "Record Key," you can look back on the Produce live and update the photo taken during the live as the Memory's photo.

※Photos set for business cards and the like cannot be updated.
※Depending on when the Memory was created, the live will be unlocked by achieving the True End, and the costume and hairstyle will be those linked to the song.
※The live performance camera settings for unit idols will be those from the Produce live when the Memory was acquired.
