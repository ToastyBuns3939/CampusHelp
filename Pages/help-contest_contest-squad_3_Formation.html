By organizing memories into the formation slots of each stage, the associated Produce Idols will be organized. One main and one sub memory can be organized per formation slot. The parameters, skill cards, and P items of the organized memories will be used in the performance battle, but the targets reflected for each are different.
You can check the formation details for each formation slot of each stage.

■Main memory
Parameters, skill cards, and P items will be reflected.

■Sub memory
Part of the parameters and skill cards will be reflected. Memories with different plan types from the main memory cannot be organized.

※Even if a skill card that cannot be duplicated is included in both the main memory and the sub memory, only one will be reflected.

■Parameters
Part of the main memory parameters and sub memory parameters will be reflected.

■Skill cards
The skill cards of the organized main/sub memory will be organized.
In addition to the skill cards of the memory, basic skill cards according to the recommended effects of the main memory will be added to the possessed skill cards.

■P-items
The P-items of the main memory will be organized.

■Stage information
The stage's judging trends and plans are displayed.
