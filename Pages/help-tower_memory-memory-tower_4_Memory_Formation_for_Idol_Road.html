In The Idol Road, you can use parameters, skill cards, and P-items that memories possess.

■ Parameters
・Effect
These parameters affect the stage’s score bonus.

■ P=Items
・Effect
Items that trigger various support effects during the stage when certain conditions are met.

■ Skill Cards
・Effect
Used in The Idol Road along with the initial cards set for each plan.
Also, the maximum number of main memories and sub memories you can equip per stage is fixed. The parameters, skill cards, and P items from the equipped memories are used in performance battles, but each applies to different targets.

■ Main Memory
Parameters, skill cards, and P-items are applied.

■ Sub Memory
Some parameters and skill cards are applied. Memories with a different plan type than the main memory cannot be equipped.
※For skill cards that cannot be duplicated, even if included in multiple memories, only one copy is applied. If enhanced skill cards are included, the enhanced version takes priority.

■ Parameters
Some parameters from the main memory and sub memory are applied.

■ Skill Cards
Skill cards from equipped main/sub memories are included.
In addition to the skill cards held by memories, basic skill cards corresponding to the recommended effects of the main memory are added to the held skill cards.

■ P-items
P-items from the main memory are equipped.
