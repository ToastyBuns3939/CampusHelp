In Contests, you can use the parameters, skill cards, and P items that Memories have.

■ Parameters
・Effect
These parameters affect the score bonus in Contests.

■ P Items
・Effect
Items that trigger various support effects within the Contest. Effects activate when conditions are met.

■ Skill Cards
・Effect
Along with the initial cards assigned for each plan, these are the skill cards used in the Contest.
For more details about Contests, please refer to the help section "Contest."
