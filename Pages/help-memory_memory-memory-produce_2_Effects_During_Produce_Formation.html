By organizing Memories at the start of Produce, various effects will occur during Produce.

■ Obtainable Skill Cards
・Effect
When organized, these are skill cards that can be obtained during Produce.

※If you organize a Memory with skill cards different from the Produce idol or plan, those skill cards cannot be obtained.
※If a Memory with a non-duplicable skill card is organized, that skill card will no longer appear during Produce. Also, if you organize two or more of the same skill card, you can only obtain one.

■ Abilities
・Effect
Various effects that support Produce. Some activate automatically, while others activate when certain conditions are met during Produce.

※Some abilities are limited to activating only for specific plans.
※Effects triggered by Memory abilities or Support abilities do not cause other Memory abilities to activate.
