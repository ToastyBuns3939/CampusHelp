This is an event where you produce the target idol in Event Mode and compete for the highest Produce Rating earned during the event period.
You can participate by turning on Event Mode from Produce during the event period. Based on your ranking in the High Score Leaderboard at the end of the event and the highest score you achieved, you can earn rewards.

During the event period, you can obtain the event item “Twinkling Stars” through Produce and other activities. “Twinkling Stars” can be used to pull from the event gacha.

Skill Cards, P-Items, and P-Drinks obtained in Event Mode will not be registered in the P-Encyclopedia if they have not been acquired before.
