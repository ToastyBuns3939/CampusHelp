When you assign Memories to each stage slot, the associated Produce Idols are also assigned.
Each slot allows you to assign both a Main and Sub Memory. The parameters, skill cards, and P Items of the assigned Memories are used, but the way they are applied differs for each.

■ Main Memory
Its parameters, skill cards, and P Items are all applied.

■ Sub Memory
A portion of its parameters and its skill cards are applied.
You cannot assign a Memory with a different Plan Type from the Main Memory.
※Skill cards that cannot be duplicated will only be applied once, even if included in both Main and Sub Memories.

■ Parameters
Parameters from the Main Memory and a portion from the Sub Memory are applied.

■ Skill Cards
Skill cards from both Main and Sub Memories are assigned.
In addition to the Memory’s own skill cards, basic skill cards based on the Main Memory’s recommended effect are also added to your deck.

■ P Items
Only the P Items from the Main Memory are assigned.

■ Stage Info
Displays each stage’s judging tendency and Plan.
