Memories can be organized for both "Produce" and "Contest," each having different effects.

Memories are generated upon completing a Produce, with effects determined by the Produce’s results.

■ When organized for Produce
By organizing Memories at the start of Produce, various effects will occur during Produce. For details, see the help section "Memories" under "Effects when organized for Produce."

■ When organized for Contest
In Contests, you can use the parameters, skill cards, and P items that Memories have. For details, see the help section "Memories" under "Effects when organized for Contest."

■ Memory Protection
By protecting a Memory, it can be excluded from conversion targets.

■ Memory Tags
Attaching tags as markers to Memories allows you to reference those tags in filters and sorting.
