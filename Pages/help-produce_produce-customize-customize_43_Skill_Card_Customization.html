Enhanced skill cards can have their effects added or strengthened through customization in Special Training.
The number of customizations and the available options are determined for each skill card.
Additionally, effects added by customization will always activate last.

■ Skill Cards That Cannot Be Customized
The following skill cards cannot be customized:
・Exclusive skill cards of the Produce idol
・Skill cards initially possessed at the start of Produce
・Trouble cards

■ Memory Draw for Customized Skill Cards
Skill cards customized during Produce may sometimes be granted to Memory with some of the applied customizations reflected.

■ Memory Synthesis
Customized skill cards cannot be selected as replacement targets for the source skill cards in Memory synthesis, and if the Memory used for synthesis holds a customized skill card, it will not appear as a replacement candidate.
