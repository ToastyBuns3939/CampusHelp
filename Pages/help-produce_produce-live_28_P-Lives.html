The content of the live performance changes depending on the final exam ranking and the affection level of the idol during production.

In a True End, you can view a special live performance.

You can take photos during the live, which are used to generate Memories.

Photos can also be saved to your photo album or device.

■ Regular Performance “Hajime” Live Details
In the regular performance *“Hajime”*, if you reach an A~D End, the live will be a “Hajime” performance for most Produce Idols.

For some Produce Idols, even with an A~D End, the live will be their P Idol song performance.

By achieving a True End, you can view a special P Live.

Live performances for each P Idol can be checked via [Shooting > Live Shooting].

■ “NEXT IDOL AUDITION” Live Details
In “NEXT IDOL AUDITION,” if you achieve a True End with a *Campus mode!!* Produce Idol, you can view a special *Campus mode!!* P Live.

If the True End has not been achieved for a *Campus mode!!* Produce Idol, you will see the A End *Campus mode!!* live.

Additionally, for non-*Campus mode!!* Produce Idols, if you set the “NEXT IDOL AUDITION Live Settings” to “P Idol Song Live Performance” and achieve a True End, you will be able to view that P Idol’s song performance.

Once you achieve a True End with a target idol in “NEXT IDOL AUDITION,” the “NEXT IDOL AUDITION Live Settings” will become available.

■ About Live Settings
After achieving a True End in “NEXT IDOL AUDITION,” you can check the “NEXT IDOL AUDITION Live Settings” via the button next to the song name before starting production.

Selecting “Campus mode!! Live Performance” will change the live to a *Campus mode!!* performance.

If you own the corresponding *Campus mode!!* Produce Idol, the *Campus mode!!* Produce Idol commu will also play.

If you select “P Idol Song Live Performance,” the live will be based on the song associated with the Produce Idol.

In this case, the Produce Idol’s own commu will play as well.
