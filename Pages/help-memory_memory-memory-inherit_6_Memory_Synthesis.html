■ Memory Synthesis
By using a Memory Synthesis Ticket, you can swap the "Stage" skill card of a Memory with candidate skill cards held by another Memory.
Memory synthesis requires two Memories: the base Memory and the Memory used for synthesis.

※The Memory used for synthesis will disappear and be lost after the synthesis.
※A Memory that has once been used as a base for synthesis cannot be used as a base Memory for synthesis again.

■ Skill Card Replacement
The candidate skill cards for replacement are randomly selected from the replacement candidates of the skill cards held by the Memory used for synthesis.

※Skill cards obtained through Memory synthesis will not exceed the rarity or enhancement level of the skill card selected on the base Memory.
※You cannot obtain a skill card with the same name through synthesis.
※Skill cards held by the Produce idol, Support cards, or customized skill cards are not candidates for synthesis.
