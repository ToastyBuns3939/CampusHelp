After producing, memories are generated based on the skill cards acquired, P items obtained, and the parameters you developed during the produce.

For more details about memories, please see the Help section "Memory" under "Memory."

■ About Memory Regeneration Tickets
“Memory Regeneration Tickets” are items that allow you to regenerate memories created during produce. You can select and acquire one memory from the originally generated memory or the regenerated ones. The number of times you can regenerate per produce is limited.

※Depending on the content of the produce, identical memories may be generated.
