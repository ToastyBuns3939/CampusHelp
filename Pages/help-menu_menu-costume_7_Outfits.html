You can set favorite costumes for each screen.
Setting a favorite costume for "Home" changes the costume/hairstyle of the idol set on the Home screen.
Setting a favorite costume for Produce allows you to select that costume as a user-defined costume when choosing an idol for Produce.
Setting a favorite costume for Live allows you to select that costume as a user-defined costume when choosing an idol for Live.
